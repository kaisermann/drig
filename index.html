Stylus

Want to change some default configuration?
Check the 'src/config.styl' or override any 'rolleiflex' attribute inside your own stylus stylesheet.

Note: if you prefer to import the stylus stylesheet, import the 'src/wrapper.styl' and do not forget to call the 'makeGrid()' mixin.

rolleiflex.baseFontSize HTML font size, for the useRem option
rolleiflex.useRem Use the rem unit instead of px
rolleiflex.autoBorderBox Set to false if your CSS already has something like * {box-sizing: border-box}
rolleiflex.build.container Should build the .container CSS?
rolleiflex.build.responsiveText Should build the .txt-breakpoint-position CSS?
rolleiflex.columns Number of columns in the grid
rolleiflex.spaces.gutter Gutter size
rolleiflex.spaces.padding .container--padded padding size
rolleiflex.breakpoints.XX.breakpoint Breakpoint min size = @media(min-width:xxx)
rolleiflex.breakpoints.XX.container Container max-width
