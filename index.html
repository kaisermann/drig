Syntax

You can choose between a declarative syntax or a simple 'bootstrapy', class oriented, syntax.

Declarative

The basics: a div with a 'row' attribute and its column childs with a 'col' attribute 


	

		--your content--
	

	

		--your other content--

Note: Oh, you don't need responsive columns? Forget about those size-breakpoint attributes, just use size="number Of Columns"!.

"Bootstrappy"


	

		--your content--
	

	

		--your other content--
*Optional .container class

Why?

"But Christian, why this attribute syntax madness?!"

Well, I'm a bit tired of the enormous workaround that is to use lots of classes to define your layout structure. Wouldn't it be great to understand our layout just by reading its HTML without mixing structure and style classes?


"But it's invalid markup!"

Although invalid markup doesn't necessarily affect SEO, I understand that some people don't want to mess with the validitation of their HTML and that's why there are two builds: declarative and class oriented!

Stylus

Want to change some default configuration?
Check the 'src/config.styl' or override any 'rolleiflex' attribute inside your own stylus stylesheet.

Note: if you prefer to import the stylus stylesheet, import the 'src/wrapper.styl' and do not forget to call the 'makeGrid()' mixin.

rolleiflex.baseFontSize 16px HTML font size, for the use.em option
rolleiflex.autoBorderBox true Set to false if your CSS already has something like * {box-sizing: border-box}



rolleiflex.use.em false Use the em unit instead of px.

Note: media queries em/rem units will ignore your HTML's font-size property, since they're are based on the browser's internal 'font-size' property.



rolleiflex.build.container true Should build the .container CSS? True or false.
rolleiflex.build.responsiveText true Should build the .txt-breakpoint-position CSS? True or False.
rolleiflex.build.declarative true Should build the grid with its declarative syntax?
rolleiflex.declarativePrefix "" If build.declarative is 'true', what should be the attributes prefix? (prefix-row, prefix-col, etc)



rolleiflex.columns 12 Number of columns in the grid



rolleiflex.spaces.gutter 8px Gutter size
rolleiflex.spaces.padding 32px .container--padded padding size



rolleiflex.breakpoints.XX.breakpoint Breakpoint max size = @media(max-width:xxx)
rolleiflex.breakpoints.XX.container Container max-width
